Trapped On Titan (August 2023 - December 2023)

I worked with four peers to create a sci-fi dungeon crawler: Trapped on Titan! Coded using Java and Android Studios. Created for Objects and Design course (CS 2340)

I worked on...
- designing characters
- implementing character select in configuration screen
- implementing the tilemap
- implementing the enemies using factory design pattern
- implementing power-ups using decorator design pattern

Special Thanks to Thetra00 for the tileset!
