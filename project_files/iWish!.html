iWish!

iWish! is a social media platform that allows users to share their wishlist! Users can create their own room that displays different items that they want as gifts. Afterwards, they can enter their friends' rooms, view their wishlist, and claim a wish. iWish! allows for a fun experience to make gift shopping easier!

This project was coded using Java for backend & dialogue and Processing for UI & animation. It was created for HackGT9, which is hosted by Georgia Tech's Hexlabs.

For this project, I worked on ...

  • designing & implementing UI details such as the background and gifts
  • implementing dialogue branches that appears on the console
  • implementing data structures that stored user's information

iWish was my first coding project I did outside of class. I wanted to create something fun and lighthearted. With two of my friends (and sheer willpower), we created a mockup of this social application. Although it wasn't completely functional, I was proud that we were able to transform our ideas into something physical and presentable.
