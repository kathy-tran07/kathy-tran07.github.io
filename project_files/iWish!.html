iWish! (October 2022)

The first project I ever did as a GT student is iWish!, a social media platform that allows users to share wishlists! Coded in Java and Processing. Created for HackGT9, which was hosted by GT Hexlabs.

I worked on...
- designing/coding the background and other UI details
- implementing dialogue within the command line terminal
- coding backend to store user's information
