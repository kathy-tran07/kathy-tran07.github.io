Trapped On Titan

Trapped on Titian is a 2D sci-fi dungeon crawler game. You, the player, are an astronaut who is traveling to a newly discovered planet. However, disaster strikes when aliens invade the spaceship! Now, you must fight off the aliens as you try to make it to the generator.

This project was created using Android Studio and coded entirely in Java. It was developed using Agile methodologies. It was made for CS 2340, also known as the Object and Design course at Georgia Tech.

For this project, I worked on ...

  • designing and creating character sprites
  • implementing character select on configuration screen
  • implementing tilemaps
  • spawning enemies using factory design pattern
  • adding power-ups using decorator

This was the first time I created a functional game and used Android Studio. Although Android Studio was hard to navigate at first, it was interesting to learn how to do full-stack development with it. It was fun to collaborate with four other peers and work together to make a fun, simple game!

Special Thanks to Thetra00 for the tileset!
