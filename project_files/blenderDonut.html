Blender Donut

An animation of a glistening, yummy 3D model of a donut spinning.

This project was created entirely in Blender.

From this project, I learned how to ...

  • model 3D objects
  • sculpt
  • use geometry nodes
  • render
  • create an animation

During my freshman year, I wanted to pick up a new hobby: 3D modeling! I found the notorious Blender Donut tutorial by BlenderGuru and decided to give it a shot. It was extremely fun to figure out how to use all the functions of Blender. It also gave me confidence to create more models in the future!
