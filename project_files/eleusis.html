Eleusis

Eleusis is a journal website that keeps track of your mood! Take daily quizzes to reflect on the day, and write about it and the end. Eleusis will keep track of your mood of the day and will show statistics of your moods. Inspired by Korean personality quizzes, Eleusis provides a simple way to self-reflect.

This project was created using React and coded in HTML, CSS, and Javascript. It was created for HackGTX, which is hosted by Georgia Tech's Hexlabs.

For this project, I worked on ...

  • designing user interface
  • implementing 2D graphics for each question
  • implementing animations for each question
  • designing answer path for the results of each quiz

I created Eleusis during the second Hackathon I intended. I was determined to create a functional project since my last Hackathon I was unable to deliver. It was also my first time using React. I was beyond proud of myself and my team that we were able to create a nice simple website.
