Eleusis (October 2023)

A daily journal that uses quizzes to keep track of your daily moods. Based on subconscious psychology. Coded using HTML, CSS, JavaScript, and React. Created for HackGTX, which is hosted by GT Hexlabs.

I worked on...
- designing the user interface
- implementing 2D graphics for each question
- designing answer path for each result of the quiz
