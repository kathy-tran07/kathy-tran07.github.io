Personal Website

My personal portfolio website (the one that you are currently on)! This website is used as a record of my programming journey.

This project was coded in HTML, CSS, and Javascript, and deployed using Github pages.

I always wanted to create my own website, but was too intimidated by the work. Then, one day I decided to give myself 24 hours to create a website and deploy it and the first version of the site was created. Of course, there was a lot of room to improve, so I have created different versions of the site. As I continue to evolve, so will this site.
