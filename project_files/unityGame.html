Basic Shooter Game

This game is a basic first-person shooter where the player is stuck in the middle of the woods. Around the player, haunted skeletons spawn around them, and they must fight them off with magical red bullets.

This project was created in Unity. It was made for the Unity workshop, which is hosted by Georgia 's Video Game Developer Club (VGDev).

From this project, I learned how to ...

  • add functionality and component to a game's objects
  • script in C++
  • spawn objects such as bullet and enemies
  • animate particles
  • build and test games

During my 3rd year as a student, I decided to join VGDev, so I could broaden my skills as a programmer. To learn more about game development, I attended their Unity workshop and created my first game on Unity! Since then, I've been working with other members of the club to create fun and more complex games.

All assets were provided by Georgia Tech's Video Game Developer Club (VGDev).
